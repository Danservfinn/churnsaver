## Description

Churn Saver automatically recovers failed subscription payments through intelligent nudges, incentives, and multi-channel communication. Reduce payment failures and increase revenue recovery with minimal effort.

### Key Features

- **Automatic Recovery**: Detects payment failures and initiates recovery workflows
- **Multi-Channel Communication**: Push notifications and direct messages
- **Smart Incentives**: Configurable free days to encourage payment updates
- **Intelligent Scheduling**: Reminders at optimal times (T+0, T+2, T+4 days)
- **Real-Time Dashboard**: Track recovery rates, cases, and revenue metrics
- **Zero Configuration**: Works out of the box with sensible defaults

### Benefits

- **Increase Revenue**: Recover 20%+ of failed payments automatically
- **Save Time**: Automated workflows eliminate manual intervention
- **Reduce Churn**: Proactive communication keeps subscribers engaged
- **Track Performance**: Real-time metrics and CSV exports for analysis

## App Store Copy

### Short Description (150 characters)
Automatically recover failed payments and reduce churn with intelligent nudges and incentives.

### Long Description (2000 characters)
Churn Saver is your automated solution for recovering failed subscription payments. When a payment fails, Churn Saver immediately:

1. **Creates a Recovery Case**: Tracks the failure and user information
2. **Sends Instant Nudge**: Notifies the user via push notification or direct message
3. **Applies Incentives**: Optionally grants free days to encourage payment updates
4. **Follows Up**: Sends reminders at strategic intervals (2 days, 4 days)
5. **Tracks Success**: Attributes recovered payments to recovery efforts

**Why Churn Saver?**

- **Proven Results**: Achieve 20%+ recovery rates in pilot programs
- **Zero Manual Work**: Fully automated workflows handle everything
- **Flexible Configuration**: Customize channels, incentives, and timing
- **Real-Time Insights**: Dashboard shows recovery metrics and performance
- **GDPR Compliant**: Secure data handling and retention policies

**Perfect For:**

- Whop communities with recurring subscriptions
- Creators who want to reduce churn automatically
- Teams looking to recover revenue without manual intervention
- Businesses tracking recovery performance metrics

**Setup is Simple:**

1. Install Churn Saver from the Whop App Store
2. Configure webhook endpoint in Whop settings
3. Customize recovery settings (optional)
4. Start recovering failed payments automatically

No coding required. Works immediately after installation.
